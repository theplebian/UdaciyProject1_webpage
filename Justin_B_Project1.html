What is HTML?

HTMLis an acronym for Hyper T ext Markup Language. HTML provides the basic underlying structure and static content of web pages. To use the internet a user makes a request, which is interpreted using http(hyper text tranfer protocol) which is sent and interpreted by a special computer called a server. A server responds to the request and replies via the internet. The results are rendered by the user's internet browser such as Microsoft Explorer, Safari, or Google Chrome.

The internet content that you see in your browser is divided into a number of html elements by the webpage designer. Elements are generally delineated by an opening and a closing html tag. The tags are important because html code is just some typed in plain text that is read by the browser. Plain text does not have the formating or organizational options that you find in your word processor. In plain text their are no tables, boxes, different fonts, or the ability to make text bold or italic. Tags in html tell the browser how to interpret content.

There are a few tags that stand alone without a closing tag. These tags are called void tage . A good example of a void tag is a tag that directs the browser to an image. Since images are not plain text, the image tag in html tells the browser where to get the image to display in the given area of the page.

Web Page Organization

HTML automatically divides content into blocks. Each block is automatically given a certain amount of space around it. Some elements, such as paragraphs exist in their own block of space. Others exist within that block. For example a block of paragraph text has space above and below it to seperate it from other elements, while bold or italic text within the paragraph do not break from other text. Elements that are automatically alotted their own block of space are called block elements, while those that are not are called inline elements.Some examples of inline tags are the anchor tag <a>, the italics tag <em>, and the paragraph tag <p>. Some examples of block tags are the division tag <div>, the paragraph tag <p>, and the form tag <form>.

The html programmer can use division tags to group content, including multiple block elements. For example, the navigation menu to the right was created as a separate block of content using division tags. A division or block of elements can be referred to as a parent element, and a sub-division or element within a divsion is called the child element of that divsion or element. A tree diagram or flow chart is sometimes made to illustrate the dependence of various organizational elements, known as a DOM (Document Object Model) for the webpage. The styling and positioning of elements on a web page can be refined by using CSS.

What is CSS?

CSS stands for Cascading Style Sheets. CSS is used to refine the position and styling of elements on a page. The divisons or classes defined in html can be used by CSS to apply styling. Styling defined for parent elements is automatically inherited by child elements. Thus styling instructions are said to cascade like a waterfall from parent elements to their child elements.

CSS instructions can be embeded within the header of the HTML document, inline with the elements themselves, or written in a seperate document. Normally, styling instructions are written in a seperate document that is loaded with the HTML code by the browser by a link provided in the header of an HTML document. The header contains instructions that are not displayed on the page, but tell the browser the title to display in the browser tab and other instructions that must be loaded in order to render the page correctly.
